# Alsa-nextbike · Online Marketing Challenge 2026

An interactive, single-file HTML pitch deck for the **Alsa-nextbike León** campaign — University of Leipzig OMC final pitch.

**▶ View it live:** open [`Alsa-nextbike-pitch.html`](Alsa-nextbike-pitch.html) in any browser (or serve the folder and open the file).

## The deck

"Night Ride" — the Nextbike brand palette (navy `#18425a`, teal `#0a7d99`, cyan `#00a8d8`) inverted onto a deep-navy canvas, with a dashed bike-route motif threading the story. 8 slides:

1. **Title** — animated route line + campaign framing
2. **Client ask** — count-up stats + a station-by-station revenue route
3. **One funnel, two riders** — TOFU/MOFU/BOFU + clickable Lucía / Carlos persona tabs
4. **Campaign plan** — count-up KPIs + a hover-interactive budget donut + calendar route
5. **Strategic messages** — flip cards revealing the real Spanish ad copy + a live iPhone push-notification mockup
6. **Limitations & contingency** — flip-to-reveal responses
7. **Six lessons** — hover-lift recommendation cards
8. **Thank you** — San Juan fireworks

## Structure

```
Alsa-nextbike-pitch.html   # self-contained deck (all CSS/JS inline)
assets/                    # real León campaign photos + logos
```

Fixed 16:9 stage (1920×1080) scaled to any viewport. Zero dependencies. Fonts: Fraunces + Satoshi (loaded via CDN).

Keep `assets/` alongside the HTML — images are referenced by relative path.
